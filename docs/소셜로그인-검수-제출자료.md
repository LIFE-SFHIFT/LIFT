# LIFT 네이버/카카오 소셜 로그인 검수 제출자료

이 문서는 네이버 로그인 사전 검수와 카카오 로그인/개인정보 동의항목 설정에 그대로 복사해 넣기 위한 제출용 문구입니다.

## 1. 서비스 기본 정보

서비스명: LIFT

서비스 URL: `https://life-shift.netlify.app`

서비스 설명:

```text
LIFT는 퇴직, 이직, 실직 등 생애전환 상황에서 사용자가 놓치기 쉬운 행정 절차, 필요 서류, 마감일, 공식 신청 링크를 정리해주는 행정 준비 서비스입니다.

사용자는 소셜 로그인 후 본인의 생애 이벤트와 최소 정보를 입력하고, 실업급여, 건강보험, 국민연금, 세금/퇴직금 등 확인이 필요한 절차를 리포트 형태로 확인할 수 있습니다. 최종 신청과 자격 판단은 정부24, 고용24, 국민건강보험공단, 국민연금공단 등 공식 기관에서 진행됩니다.
```

개인정보처리방침 URL: `https://life-shift.netlify.app/privacy`

서비스 이용약관 URL: `https://life-shift.netlify.app/terms`

로그인 화면 URL: `https://life-shift.netlify.app/login`

## 2. 네이버 개발자센터 입력값

서비스 URL:

```text
https://life-shift.netlify.app
```

Callback URL:

```text
https://life-shift.netlify.app/login/callback/naver
```

검수 신청 설명:

```text
LIFT는 퇴직, 이직, 실직 등 생애전환 상황에서 사용자가 놓치기 쉬운 행정 절차, 필요 서류, 마감일, 공식 신청 링크를 정리해주는 행정 준비 서비스입니다.

네이버 로그인은 회원 식별, 로그인 유지, 리포트 생성 내역 관리, 고객 문의 대응을 위해 사용합니다. 네이버 로그인 사용자는 LIFT의 별도 비밀번호를 입력하지 않으며, 네이버 로그인 완료 후 서비스 약관과 개인정보처리방침에 동의하고 생애 이벤트를 선택해 리포트를 생성할 수 있습니다.
```

제공 정보 활용처 설명:

```text
이용자 식별자: 네이버 로그인 사용자를 LIFT 회원으로 식별하고, 리포트 생성 내역과 서비스 이용 상태를 연결하기 위해 사용합니다.

이메일: 회원 문의 대응, 계정 식별 보조, 서비스 이용 관련 안내를 위해 사용합니다.

닉네임: 서비스 화면에서 사용자를 표시하고, 마이페이지/커뮤니티 등 사용자 경험을 제공하기 위해 사용합니다.
```

네이버 검수 자료 캡처 목록:

```text
1. https://life-shift.netlify.app/login 접속 화면
2. 네이버로 시작하기 버튼 클릭 화면
3. 네이버 로그인/동의 화면
4. 로그인 완료 후 LIFT 온보딩 화면
5. 서비스 이용약관 및 개인정보처리방침 동의 화면
6. 생애 이벤트 선택 화면
7. 리포트 생성 또는 리포트 미리보기 화면
8. https://life-shift.netlify.app/terms 화면
9. https://life-shift.netlify.app/privacy 화면
```

주의:

```text
네이버 검수 신청 전, 네이버 애플리케이션의 로고/앱 이름/서비스명이 실제 배포 서비스의 LIFT 표시와 일치해야 합니다.
```

## 4. 서비스 이용 흐름

검수자가 확인할 수 있는 사용자 흐름:

```text
1. 사용자가 https://life-shift.netlify.app/login 에 접속합니다.
2. 네이버로 시작하기 또는 카카오로 시작하기 버튼을 클릭합니다.
3. 네이버/카카오 로그인 및 정보 제공 동의 화면에서 동의합니다.
4. LIFT로 돌아오면 약관 동의와 생애 이벤트 선택 화면이 표시됩니다.
5. 사용자가 퇴직, 이직, 실직 중 하나를 선택합니다.
6. 최소 정보를 입력하면 행정 준비 리포트 또는 미리보기 화면에서 필요한 절차, 필요 서류, 마감일, 공식 신청 링크를 확인할 수 있습니다.
```

## 5. 반려 방지 체크리스트

- [ ] 네이버 서비스 URL: `https://life-shift.netlify.app`
- [ ] 네이버 Callback URL: `https://life-shift.netlify.app/login/callback/naver`
- [ ] 카카오 Web 사이트 도메인: `https://life-shift.netlify.app`
- [ ] 카카오 Redirect URI: `https://life-shift.netlify.app/login/callback/kakao`
- [ ] Render 환경변수 `OAUTH_MOCK_ENABLED=false`
- [ ] Render 환경변수 `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET` 입력
- [ ] Render 환경변수 `KAKAO_CLIENT_ID` 입력
- [ ] 카카오 Client Secret을 활성화했다면 Render 환경변수 `KAKAO_CLIENT_SECRET` 입력
- [ ] Netlify 환경변수 `NEXT_PUBLIC_API_BASE_URL=https://lift-api.onrender.com`
- [ ] `/terms`가 200으로 열림
- [ ] `/privacy`가 200으로 열림
- [ ] 소셜 로그인 사용자가 LIFT 별도 비밀번호를 입력하지 않음
- [ ] 이메일/닉네임을 요청한다면 실제 활용처를 제출 자료에 설명함
- [ ] 로고, 앱 이름, 서비스명이 LIFT로 일치함
